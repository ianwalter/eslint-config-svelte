# @ianwalter/eslint-config-svelte
> Ian's ESLint configuration for [Svelte][svelteUrl]

[![npm page][npmImage]][npmUrl]
[![CI][ciImage]][ciUrl]

## Installation

```console
yarn add @ianwalter/eslint-config-svelte --dev
```

## Usage


```js
  "eslintConfig": {
    "root": true,
    "extends": [
      "@ianwalter/svelte"
    ]
  }
```
